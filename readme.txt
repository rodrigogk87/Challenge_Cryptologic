Para correr el programa:

1. instalar las dependecias con npm i

2. crear un .env en el root del proyecto y configurarlo con los siguientes datos: 
    DB_URL=URL_DE_DASE_DE_DATOS_MONGO_DB, por ejemplo mongodb://localhost:27017/ChallengeCryptologic
    POLYGON_ENDOINT=https://matic-mainnet.chainstacklabs.com

2. correr el deamon de mongodb

3. correr el comando npm run program
